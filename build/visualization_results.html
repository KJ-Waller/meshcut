üîç Implementation Details:

This test demonstrates MeshCut's core algorithm:

  • Grid Classification: Each grid cell is classified as FULL_IN, FULL_OUT, or BOUNDARY
  • FULL_IN cells: Generate standard grid triangles directly (efficient)
  • BOUNDARY cells: Use Sutherland-Hodgman clipping + Earcut triangulation
  • Vertex Deduplication: Shared vertices across cell boundaries create coherent mesh
